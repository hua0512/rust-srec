# 更新日志

## `unreleased`

### 修复

- **管道执行详情在移动端正常显示**

  在窄屏上，管道执行详情页的概览卡片可能出现图标重叠、进度百分比等较长数值被截断的情况。现在这些卡片会自适应小屏幕，进度、步骤数和开始时间都能正常显示。

- **主题更改重新即时生效**

  此前如果你已经自定义过主题，在主题页面选择新主题后需要刷新页面才能生效。现在主题预设、颜色和圆角的更改都会立即生效。

- **列表的筛选和搜索在返回时会保留**

  在主播、会话、管道任务、预设、工作流、媒体产物等页面上，此前只要打开某一项再返回、或刷新页面，你的搜索内容、筛选、排序和所在页码都会被重置。现在这些状态会保存在页面地址中，返回或刷新都会停留在原处，你还可以收藏或分享某个筛选后的视图。

- **分页按钮现在会跟随界面语言**

  在分页列表中，“上一页”和“下一页”按钮此前即使界面切换到其他语言仍显示英文。现在它们会以你选择的语言显示。
