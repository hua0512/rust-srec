# 更新日志

在这里查看 rust-srec 的版本更新、行为变更、兼容性说明与升级建议。

## 未发布

- [`unreleased`](./unreleased.md) — 暂无下一个版本的待发布改动

## 最新版本

- [`v0.4.0`](./v0.4.0.md) — 新增 SOOP 与 Bigo Live 两个平台、重写的 Mesio HLS 录制引擎、主播批量操作与列表筛选、数据库自动清理，以及全新的无闪烁主题系统，另含录制回放与流水线编辑器修复

## 历史版本

- [`v0.3.2`](./v0.3.2.md) — 管道与录制可靠性改进：结束阶段与配对后处理会等待所需文件就绪，恢复录制后分段编号保持连续，**删除源文件** 不再删除已转换的视频，CDN 临时故障也不会再把直播中的主播误标为离线

- [`v0.3.1`](./v0.3.1.md) — 延续 v0.3.0 的录制会话稳定性改进（降低临时下载失败误结束会话、更干净的排程结束、更安静的排程外重复检查），并为系统健康页面新增 GPU 健康监测，`/api/health` 端点性能与资源占用也大幅优化

- [`v0.3.0`](./v0.3.0.md) — 输出根写入门（#508）、录制会话生命周期重构、带"排队中"徽章的优先级感知队列、主播检查历史条、rclone 带宽与吞吐量控制、后端通知本地化、Mesio `--disable-log-file`，以及一批 Huya 与小红书平台修复

- [`v0.2.1`](./v0.2.1.md) — 录制正确性修复、会话生命周期改进、Douyu 更新、弹幕统计增强与前端稳定性提升

后续发布新版本后，会在这里继续归档更早的更新日志。
