# 抖音

[抖音](https://www.douyin.com) 是中国领先的短视频和直播平台。

## URL 格式

```
https://live.douyin.com/{房间号}
```

## 功能

- ✅ FLV 和 HLS 流
- ✅ 弹幕采集
- ✅ 多画质选项
- ✅ 可选择 PC 或 移动端 API
- ✅ 互动玩法（互动直播）检测

## 认证

抖音通常不需要配置 Cookie 即可正常记录。如有特殊需要，您仍可在配置中设置 `ttwid` 等 Cookie。

在 **设置 (Settings)** → **平台 (Platform)** → **抖音 (Platform-douyin)** 中进行配置。

## 注意事项

- **流格式选择**：强烈建议优先录制 **FLV** 格式，以获得更好的稳定性。
- **画质选择**：可在配置中开启 `force_origin_quality` 以强制获取最高画质。（实验性：如果所请求的画质不可用，可能导致无法获取视频流）。
- **双画面**：默认支持双画面数据采集。
- **互动玩法**：可配置自动跳过“互动玩法”（互动直播）的录制。
- **移动端 API**：如果 PC 端接口失效，可以尝试开启 `force_mobile_api`。
- **地区限制**：部分直播有地区限制，可能需要中国大陆代理。
- **不支持的内容**：目前不支持 **电台直播** (音频流) 的录制。
