# Bigo Live

[Bigo Live](https://www.bigo.tv) 是面向全球创作者的直播平台。

## URL 格式

```
https://www.bigo.tv/{site_id}
https://www.bigo.tv/{locale}/{site_id}
```

示例：

- `https://www.bigo.tv/221338632`
- `https://www.bigo.tv/ja/221338632`
- 用户名短链如 `https://www.bigo.tv/username`

## 功能

- ✅ HLS 直播（公开接口通常只有单一清晰度 `live`，多为竖屏约 540p）
- ✅ 弹幕采集（游客 WebSocket：公屏聊天与礼物）
- ✅ 密码锁房
- ✅ 默认签发与网页一致的完整性 token（可关闭）
- ❌ 公开接口下的多清晰度 / 多码率选择
- ❌ 账号登录或仅网页播放器才有的更高清晰度（如 1080p HEVC）

## 配置

平台选项在 **设置** → **平台** → **Bigo**。**公开房间无需任何配置**，添加 URL 即可录制。

| 选项 | 默认 | 说明 |
|------|------|------|
| **直播间密码**（`stream_password`） | 空 | 密码锁房的默认密码。可在主播 URL 上用 `?pwd=...` 覆盖。 |
| **签发完整性 token**（`mint_token`） | 开启 | 向 studio 请求附带与网页类似的完整性 token。仅在你所在网络签发失败时关闭；多数公开房间关闭后仍可录制。 |

### 密码锁房

1. 在平台（或主播）配置中填写 `stream_password`，**或**
2. 在主播 URL 后追加 `?pwd=密码`（URL 优先于平台默认）。

未提供密码时，锁房会以私密内容错误失败。

### 弹幕

为主播开启弹幕后：

- **会记录**：公屏聊天与礼物（游客 WebSocket，无需账号）。
- **不支持**：发送弹幕、进房提示、点赞、关注/分享提示。

::: info
- **清晰度**：公开接口每个房间只返回一条媒体播放列表（标签为 `live`），一般为竖屏中等清晰度，大约 **480p–540p**（例如约 480×854 的 H.264）。此路径没有多码率列表，完整性 token 也不会解锁更高的公开清晰度。浏览器侧偶见的 1080p HEVC 不通过该接口提供。
- **认证**：录制与接收弹幕均不需要 Cookie 或 OAuth。
- **网络**：若完整性 token 签发被拦截，关闭 **签发完整性 token** 后重试。
:::
